# Galley Epoque — Website UI Kit

## Overview
High-fidelity click-through prototype of the Galley Epoque public website.

## Screens
1. **Homepage** — Hero, featured works, gallery grid, about strip, contact CTA
2. **Gallery Grid** — Filterable grid of artworks
3. **Artwork Detail** — Full artwork view with purchase/contact action
4. **Contact** — Enquiry form

## Component Files
- `Header.jsx` — sticky navigation bar
- `HeroSection.jsx` — full-bleed hero with headline + CTA
- `GalleryGrid.jsx` — filterable grid with artwork cards
- `ArtworkCard.jsx` — individual artwork card
- `ArtworkDetail.jsx` — detail/lightbox overlay
- `ContactSection.jsx` — enquiry form section
- `Footer.jsx` — site footer

## Design Width
1440px desktop, responsive down to 375px mobile.
